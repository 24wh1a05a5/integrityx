# 🚀 IntegrityX



IntegrityX is an automated assignment evaluation system built around GitHub workflows.  
Instead of submitting code through traditional portals, students push their code to repositories, and the system evaluates it automatically.

---

## 💡 What makes it useful?

- No manual code checking  
- Instant feedback after every push  
- Fair and consistent evaluation  
- Uses real-world tools like GitHub  

---

## ⚙️ How it works

1. Student pushes code to GitHub  
2. A webhook is triggered  
3. Backend fetches the latest code  
4. Predefined test cases are executed  
5. Results are calculated  
6. Leaderboard updates automatically  

---

## 📊 Output

- Number of test cases passed  
- Ranking based on performance  
- Real-time leaderboard updates
